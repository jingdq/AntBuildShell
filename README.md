AntBuildShell
=============

Linux bash shell auto build Android Project.

#How to use#
step1:

git clone https://github.com/clarck/AntBuildShell.git AntBuildShell

step2:

  Run the following command:

  cd AntBuildShell

  sh build.sh

#Output file#
output file path:

  publish_apks/date_time_svnreversion_release.apk
